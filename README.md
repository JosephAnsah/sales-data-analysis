# 📊 Sales Data Analysis Project

## 📌 Overview
This project performs end-to-end sales data analysis using Python.

It covers:
- Data cleaning
- Feature engineering
- Data aggregation
- Business insights generation

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy

## 🔍 Key Analysis Performed

- Revenue by product
- Revenue by region
- Market share analysis
- Top-performing transaction identification

## 💡 Key Insights
- Identified the top-performing product
- Determined highest revenue region
- Analyzed contribution of each product to total revenue
